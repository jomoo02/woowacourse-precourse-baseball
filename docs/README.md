# 구현할 기능 목록
- [ ] 유저
  - [ ] 세 자리의 숫자를 받아 배열의 형태로 저장하고 반환한다.
    - [ ] 세 자리의 숫자는 서로 다른 수여야 한다.
    - [ ] 1 ~ 9 사이의 양의 정수들로 구성되어야 한다.
    - [ ] 공백이 없어야 한다.
- [ ] 컴퓨터
  - [ ] 유저 객체를 상속받는다.
  - [ ] 3개의 숫자를 받아 저장되어 있는 숫자와 비교해 결과를 반환한다.
- [ ] 입력 뷰
  - [ ] 숫자를 입력받는다.
  - [ ] 게임 새로 시작 여부를 입력 받는다.
    - 새로 시작 하려면 1, 종료하려면 2
- [ ] 출력 뷰
  - [ ] 매치 결과를 출력한다 
    - ex: 1볼 1스트라이크, 3스트라이크, 낫싱
  - [ ] 3스트라이크면 게임 종료를 알린다.
